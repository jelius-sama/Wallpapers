# 🎨 Hyprland Wallpapers

A personal collection of wallpapers that look great on **tiling window managers**, especially **Hyprland**.

These wallpapers are selected to work well with **minimal setups, Waybar themes, and modern Linux desktops**.

---

## 📦 Repository Size

⚠️ This repository is **large (~1 GB)** because most wallpapers are **high-resolution (2K / 4K)**.

If you are using a **metered or slow connection**, consider downloading **individual images** instead of cloning the entire repository.

---

## 🖼️ Image Sources

This repository contains wallpapers collected from **different sources**:

* 📸 **Most wallpapers are curated high-quality images** from public wallpaper collections.
* 🤖 **A small number of wallpapers are AI-generated.**


## 📥 Installation

Clone the repository to your home directory:

```bash
git clone --depth=1 https://github.com/Ajaymanikandan0x/hyprland_wallpapers.git ~/hyprland_wallpapers
cd ~/hyprland_wallpapers
```
